## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/shadabtanjeed/IT-Tasks.git
   ```
2. Navigate to the project directory:
   ```bash
   cd IT-Tasks/2_id_card_generator
   ```
3. Install the dependencies:
   ```bash
   flutter pub get
   ```
4. Run the application:
   ```bash
   flutter run
   ```
